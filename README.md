# 古籍竖排繁‧简对照生成工具

一款用于将古籍竖排繁体文字转换为简繁对照格式的工具。

## 项目简介

本工具旨在帮助研究者和爱好者更便捷地阅读古籍文献，通过OCR技术识别竖排繁体文字，并自动生成简体对照版本，提升阅读体验和理解效率。

## 主要功能

- 支持PDF格式古籍文献的处理
- 自动检测并分割文本列
- 竖排繁体文字OCR识别
- 繁体到简体的智能转换
- 生成简繁对照的PDF文档

## 使用方法

```bash
# 安装依赖
pip install -r requirements.txt

# 处理单个PDF文件
boocr poc --input 古籍文件.pdf --output 输出文件.pdf
```
